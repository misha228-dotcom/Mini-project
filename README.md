# Mini-project: Аксолотль

Сайт створений для ознайомлення з аксолотлем — унікальною водною твариною.

## На сайті є:

* інформація про аксолотля;
* цікаві факти;
* особливості харчування;
* корисні посилання;
* форма опитування.

## Технології

* HTML5
* CSS3

## Мета проєкту

Розповісти про аксолотля та його унікальні особливості.
